Lesson 1: Introduction to HTML

What is HTML?

HTML (HyperText Markup Language) is the fundamental language for creating web pages and web applications. It defines the structure of content on the web. HTML elements, represented by tags, define different types of content such as headings, paragraphs, links, images, and more.

Key Concepts in HTML

Here are some important concepts and features of HTML:

  • Tags: HTML is made up of elements represented by tags. Tags usually come in pairs like <tag> and </tag>, enclosing the content between them.
  • Attributes: Tags can have attributes that provide additional information about an element. For example, the <a> tag has an attribute href that defines the link's destination.
  • Nesting: HTML elements can be nested inside one another to create more complex structures. For example, a list can have list items inside it.
  • Comments: You can add comments in HTML code using <!-- comment here -->. Comments are ignored by the browser.

Basic Structure of an HTML Document

An HTML document follows a basic structure that includes several essential elements:

<!DOCTYPE html>  
<html lang="en"> 
  <head>
    <meta charset="UTF-8">  
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
    <title>Page Title</title> 
  </head>
  <body>
    <h1>This is a heading</h1>  
    <p>This is a paragraph.</p>  
    <a href="https://example.com">Visit Example</a> 
  </body>
</html>

Common HTML Tags and Their Usage

Here are a few of the most commonly used HTML tags:

  • <h1> to <h6>: These are heading tags, with <h1> being the largest and most important, and <h6> the smallest.
  • <p>: Defines a paragraph. Itâ€™s a block-level element that takes up the full width available.
  • <a>: Creates a hyperlink. The href attribute defines the URL.
  • <img>: Embeds an image. The src attribute specifies the image file path.
  • <div>: A division or section in the document. It groups other elements.

HTML5 Features

HTML5 introduced several new elements and features for building modern websites:

  • <header>, <footer>, <section>, <article>, <nav>: These semantic elements help in structuring the content logically and improve accessibility.
  • Multimedia Support: HTML5 supports embedding audio (<audio>) and video (<video>) content without needing external plugins like Flash.
  • Canvas and SVG: Allows for the drawing of graphics and animations directly in the browser using <canvas> or Scalable Vector Graphics (SVG).

Lesson 5: JavaScript Basics

What is JavaScript?

JavaScript is a versatile programming language used to add interactivity, control multimedia, animate images, and perform various other dynamic tasks on web pages. Unlike HTML and CSS, which define the structure and style of a page, JavaScript makes the page come alive.

Basic JavaScript Syntax

Here is a basic function in JavaScript:

function greet() {
  alert('Hello, welcome to the website!');
}

JavaScript in HTML

JavaScript can be embedded directly in HTML using the <script> tag. When placed in the body section, it can interact with elements on the page.

Conditions

Conditional statements control how your code behaves based on different conditions. For example, an if-else statement:

let time = 15;

if (time < 12) {
  alert('Good morning!');
} else {
  alert('Good afternoon!');
}

Loops

Loops are used to execute a block of code multiple times. Below is an example of a for loop:

for (let i = 0; i < 5; i++) {
  console.log('This is loop number ' + i);
}

Lesson 2: HTML Tags

Understanding HTML Tags

HTML tags are the building blocks of web pages. They define elements that structure the page and add content like text, images, links, and more. Each HTML tag usually comes in pairs: an opening tag <tag> and a closing tag </tag>.

Common HTML Tags

  • <h1> to <h6>: Used for headings. <h1> represents the main heading (largest) and <h6> the least important (smallest).
  • <p>: Represents a paragraph. It is a block-level element and takes up the full width available.
  • <a>: Defines a hyperlink, allowing users to click and navigate to another page or resource. The href attribute specifies the destination URL.
  • <img>: Embeds an image on the page. The src attribute is used to provide the path to the image file.
  • <ul> and <li>: <ul> creates an unordered (bulleted) list, and <li> represents an individual list item.
  • <ol> and <li>: Similar to <ul>, <ol> creates an ordered (numbered) list, while <li> defines the list items.
  • <div>: A container used to group elements together for styling or layout purposes. It does not add any visual change by itself.
  • <span>: A container like <div>, but it's an inline element used to group content without breaking the flow of text.

Example of Basic HTML Tags

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>
<a href="https://example.com">This is a link</a>
<img src="image.jpg" alt="Description of the image">
<ul>
    <li>List item 1</li>
    <li>List item 2</li>
</ul>

HTML Attributes

Attributes provide additional information about HTML elements. They are written inside the opening tag and typically come in name-value pairs like name="value". Some common attributes include:

  • href: Specifies the destination of a link in the <a> tag.
  • src: Defines the source of an image in the <img> tag.
  • alt: Provides alternative text for an image if it can't be displayed.
  • class: Assigns a CSS class to the element for styling purposes.
  • id: Provides a unique identifier for an element, used in JavaScript or CSS.

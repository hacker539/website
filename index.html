    • 1.First lesson

    →In this lesson, we'll learn what HTML is:
    HTML is a code that tells a web browser how to display text, images, videos, and other content.
    HTML stands for HyperText Markup Language. When you open a webpage in your browser, you're viewing the HTML code for that page rendered into a visual format.
    HTML is made up of elements, which are like the building blocks of a webpage. Each element tells the browser what kind of content it should display and how it should be formatted. For example, the <p> element creates a paragraph, the <h1> element creates a heading, and the <img> element creates an image.

    ∘Features of HTML∘
    It is platform-independent. For example, Chrome displays the same pages identically across different operating systems such as Mac, Linux, and Windows. Images, videos, and audio can be added to a web page (For example - YouTube shows videos on their website) HTML is a markup language and not a programming language. It can be integrated with other languages like CSS, JavaScript, etc. to show interactive (or dynamic) web pages

    →If you are using VS Code for programming, there is a shortcut you can use to automatically generate a draft body of HTML. The shortcut is to type "!" and then hit enter. This will generate the basic structure of an HTML document, including the <html>, <head>, and <body> tags. This can save you a lot of time when you are starting a new web project.

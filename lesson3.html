Lesson 3: CSS Basics

What is CSS?

CSS (Cascading Style Sheets) is a language used to control the presentation of HTML documents. It allows you to style elements on a web page, making them visually appealing and adaptable to different screen sizes. CSS is essential for web design and is used to define layout, colors, fonts, and more.

Why CSS is Important

CSS separates content (HTML) from presentation, allowing you to easily apply consistent styles across a website. It also provides flexibility and control over how your pages look on various devices, such as mobile phones, tablets, and desktops.

Basic CSS Syntax

CSS is composed of selectors, properties, and values. Here's the general structure:

selector {
  property: value;
}
  • Selector: The HTML element you want to style (e.g., body, h1, p).
  • Property: The aspect of the element you want to change (e.g., color, font-size, margin).
  • Value: The specific style you want to apply to the property (e.g., red, 16px, 10px).

Types of CSS

CSS can be applied in three ways:

  • Inline CSS: CSS written directly inside an HTML tag using the style attribute.
  • Internal CSS: CSS placed within a <style> tag in the head of your HTML document.
  • External CSS: CSS stored in a separate file (e.g., styles.css) and linked to the HTML file using a <link> tag.

Example of an External CSS Link

<link rel="stylesheet" href="styles.css">

This example links an external CSS file to your HTML document, keeping your design and content separate for better organization and maintainability.
